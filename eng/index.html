What is myboar?

MyBoar is a simple file sharer that uses CDNs.

Creating a server is a easy procedure, but you need a license

MyBoar allow you to chat, allow you to share news, allow you to share links and many other things... cmon, try it

Help

Create a server

  1. Get a myboar license from TSAZ
  2. Go back to myboar and insert the key
  3. Configure your server
  4. Create and enjoy!!

    5. Need help?

Share a file

  1. Download your file from the browser
  2. Go to the download section and get the Download URL
  3. From this url take host (www.ex.com) protocol (HTTPS) and path (/file.png)
  4. Configure the file on the application and enjoy!


Security warning

Stay secure! TSAZ dont check files, dont download file that you dont know (like executables files)
