Cos'e' MyBoar?

MyBoar e' un semplice condivisore di file che utilizza i CDN.

Creare un server e' una procedura semplice, ma e' necessaria una licenza.

MyBoar ti permette di chattare, condividere notizie, condividere link e molto altro... Dai, provaci!

Aiuto

Come creare un server

  1. Acquista una licenza MyBoar da TSAZ
  2. Torna a MyBoar e inserisci la chiave
  3. Configura il tuo server
  4. Crea e divertiti!!

    5. Hai bisogno di aiuto?

Come condividere un file

  1. Scarica il file dal browser
  2. Vai alla sezione Download e ottieni l'URL di download
  3. Dall'URL, prendi l'host (www.ex.com), il protocollo (HTTPS) e il percorso (/file.png)
  4. Configura il file nell'applicazione e goditi!
